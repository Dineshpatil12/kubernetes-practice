# Kubernetes Practice Repository

This repository contains Kubernetes hands-on labs completed for DevOps learning and interview preparation.

## Repository Structure

```text
kubernetes/
├── level-1/
└── level-2/
```

## Level 1

Completed foundational Kubernetes labs:

* Pods
* Deployments
* ReplicaSets
* Namespaces
* Resource Limits
* Rolling Updates
* Rollbacks
* Jobs
* CronJobs
* Services
* NodePort
* Volume Troubleshooting

## Level 2

Advanced Kubernetes labs:

* Shared Volumes
* ConfigMaps
* Secrets
* Init Containers
* Sidecar Containers
* Persistent Volumes
* Persistent Volume Claims
* Storage Troubleshooting

## Objective

Build practical Kubernetes administration and troubleshooting skills through hands-on exercises and real-world scenarios.
